# Greenden-Tailwind
# 🌿 Greenden - Responsive Plant Store Website

A modern, elegant, and fully responsive **Plant Store Website** built using **HTML5**, **Tailwind CSS**, and **JavaScript**. The project focuses on creating a clean and user-friendly interface for an online plant shopping experience while ensuring responsiveness across all devices.

---

## 🌐 Live Demo

🔗 **Website:** https://ru942.github.io/Greenden-Tailwind/

---

## 📌 Project Overview

Greenden is a responsive front-end web application designed for a fictional plant store. The website showcases a premium collection of indoor and outdoor plants with an attractive and modern UI.

The project demonstrates responsive web design principles using **Tailwind CSS**, interactive navigation using **JavaScript**, and clean HTML structure.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ru942/Greenden-Tailwind.git
```

### Navigate to the Project Folder

```bash
cd Greenden-Tailwind
```

### Open the Project

Simply open **index.html** in your browser or use **Live Server** in Visual Studio Code.
